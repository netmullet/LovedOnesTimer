# みんなの寿命をカウントダウン - みん寿 | LovedOnesTimer

**残りの時間を大切にするための寿命カウントダウンアプリ**

## 概要
みん寿は、自分や大切な人の想定寿命に基づいて残りの日数を計算・表示するiOSアプリケーションです。時間の有限性を意識することで、今この瞬間をより大切に過ごすためのツールとして開発しました。
- [App Store Link](https://apple.co/4pgdwxX)

### 開発した背景
大切な人を失ったときに「もっと会っておけばよかった」と後悔した経験がありました。そこから、変化の激しい現代を生きる人は、目の前のことに追われ、日々を忙しく過ごすあまり、人生という時間の有限性をつい忘れてしまいがちなのではないかと考えました。人生の残り時間を意識することで、日々を漫然と過ごすのではなく、少しでも有意義に過ごすきっかけになるものを作りたいという思いで、開発しました。
### 工夫した点
ユーザー体験の質を高めるため、AppleのHuman Interface Guidelinesを参照し、iOSの標準的なUIパターンに沿ったデザインを意識しました。また、タッチターゲットを指が届きやすい位置に配置するなど、ユーザーが直感的に操作できることを重視しました。
### 苦労した点
Widgetの実装において、アプリ本体とWidget間でデータを共有する実装に苦戦しました。App Groupsを使った共有コンテナの設定と実装方法が分かるまで時間がかかりましたが、Stack Overflowや海外の技術ブログで情報を収集し、仮説と検証を繰り返すことで解決しました。

## 主な機能

### コア機能
- **寿命のカウントダウン**: 誕生日と想定寿命から想定寿命までの残り日数を算出し、カウントダウン
- **ウィジェット対応**: ホーム画面からカウントダウンを常時確認
